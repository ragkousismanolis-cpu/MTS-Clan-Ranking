Μαζικό Import Matches

Κάνε paste οριζόντια από Excel / Google Sheets ή ανέβασε εικόνες scoreboard. Το bulk import διαβάζει τις γραμμές ανά 2, άρα 2 γραμμές = 1 official match.

      Ιστορικό Official Matches

      Προσθήκη / Επεξεργασία Παίκτη

      Μαζικό Import Παικτών

      Κάνε paste από Excel / Google Sheets με 2 στήλες: USERNAME και Starting ELO. Δέχεται tab, ελληνικό ή αγγλικό separator, και μία γραμμή ανά παίκτη.

      Αν υπάρχει ήδη παίκτης με ίδιο όνομα, θα γίνει update μόνο στο Base ELO του και θα μείνουν ίδια τα υπόλοιπα starting stats.
      Αν σβήσεις παίκτη που υπάρχει ήδη σε match, τα παλιά matches μένουν αποθηκευμένα αλλά ο παίκτης δεν θα εμφανίζεται ξανά στις επιλογές.

      Known Issues

      Πώς δουλεύει το ranking

      • Μπορείς να κάνεις edit/delete οποιοδήποτε match.
      • Με import/export κρατάς backup χωρίς server.
      • Μπορείς να κρύβεις παίκτες από το leaderboard χωρίς να τους αφαιρείς από matches.
      • Τα δεδομένα αποθηκεύονται τοπικά στον browser μέσω localStorage.
